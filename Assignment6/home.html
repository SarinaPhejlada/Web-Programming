Part A

An employee log program that allows the user to input the amount of hours each employee has worked until a -1 is entered. After the hours are entered, the user can select the "Create Table" button to generate a table that displays how many employees were entered, how many hours each employee worked, and what each of their pay is(including overtime). Last it prints the total amount paid towards the employees.

Part B

A simple number guessing game. The program generates a random number and the user has 10 tries to guess the secret number.

Part C

A picture matching game. The user gets to choose their level of difficulty before clicking "Start". Once the game starts, a timer is set and the user has BLANK seconds to try and match all the pictures.
